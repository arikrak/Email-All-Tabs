Email-all-tabs

A chrome extension which lets you email all open tabs, while converting long URLs to short ones.

Download this project as a .zip file Download this project as a tar.gz file

Email All Tabs allows you to email the URL of all tabs in current window to someone (or yourself). Instead of having to copy and paste each link, you can send them all with one click. This extension can be more effective when combined with a tab-managing extension such as "Tab Scissors".
